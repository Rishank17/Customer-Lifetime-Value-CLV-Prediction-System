# Customer Lifetime Value (CLV) Prediction System

## Overview

This project predicts Customer Lifetime Value (CLV) using Machine Learning and visualizes customer insights using Power BI.

## Features

- Data Cleaning
- Feature Engineering
- RFM Analysis
- Customer Segmentation
- Random Forest Regression
- Power BI Dashboard

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Power BI

## Machine Learning

Model Used:
- Random Forest Regressor

Evaluation Metrics:

- MAE: 2091.25
- R² Score: 0.576

## Dashboard Features

- Total Customers
- Total Revenue
- Average CLV
- Customer Segmentation
- Revenue by Segment
- Top Customers
- Customer Behavior Analysis
